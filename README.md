# MonaLingua — Language Translation Tool

A simple, browser-based language translation tool built as part of the **CodeAlpha Artificial Intelligence Internship** (Task 1).

Powered by **Monalisa Inc. · INDIA**

## What it does

- Enter any text and pick a source and target language
- Get an instant translation between 28 supported languages, including English, Spanish, French, German, Hindi, Bengali, Tamil, Telugu, Arabic, Chinese, Japanese, and more
- Swap the source and target languages with one click
- Listen to the source text or the translation read aloud (text-to-speech)
- Copy the translated text to your clipboard with one click

## How it works

- Pure **HTML, CSS, and JavaScript** — no build step, no installation required
- Translation is powered by the free [MyMemory Translation API](https://mymemory.translated.net/)
- Text-to-speech uses the browser's built-in Web Speech API

## How to run it

1. Download `index.html` from this repository
2. Open it in any modern web browser (Chrome, Edge, Firefox, Safari)
3. Enter text, choose your languages, and click **Translate**

No server, API key, or installation needed — it runs entirely client-side, as long as you're connected to the internet.

## Tech stack

- HTML5 / CSS3
- Vanilla JavaScript (Fetch API, Web Speech API, Clipboard API)
- MyMemory Translation API
